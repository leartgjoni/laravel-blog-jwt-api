# Laravel Blog JWT API

This is a blog api with jwt auth that communicates with a react native app. Both demos are linked below<br/>

# How to run the app
	composer install
	php artisan migrate
	php artisan serve

# Demo
Web: http://chat-laravel-redis.lersoft.al
